Checklist de Segurança da Informação

Avalie sua aderência aos frameworks ISO 27001, LGPD, PCI DSS, SOC 2, DevSecOps, Finanças TI e Governança de IA. Seus dados são salvos localmente no navegador. OK = atendido; Não OK = não atendido; N/A = não aplicável. Aderência ≥ 80% = Aderente.

Como usar: Em cada aba, marque OK, Não OK ou N/A para cada item. O percentual é calculado como OK ÷ (OK + Não OK). Itens vazios e N/A são excluídos. Referências: ISO/IEC 27001:2022 Annex A, LGPD (Lei 13.709/2018), PCI DSS v4.0, SOC 2 (AICPA TSC), OWASP DevSecOps Guideline, FinOps/ITFM, Framework AIE (Gov.br), OSIC 15/2024.
